JavaScript отключен

Для работы нашего приложения необходимо включить JavaScript в настройках браузера.

Как включить JavaScript:

  • Chrome: Настройки → Конфиденциальность и безопасность → Настройки сайта → JavaScript → Разрешить
  • Firefox: Настройки → Приватность и защита → прокрутите вниз и найдите JavaScript
  • Edge: Настройки → Файлы cookie и разрешения сайта → JavaScript
  • Safari: Настройки → Безопасность → Включить JavaScript

После включения JavaScript обновите страницу.

Возможно, JavaScript блокируется расширениями браузера (AdBlock, NoScript и т.д.)
